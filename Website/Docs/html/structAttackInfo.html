Detailed Description

Struct container, describing an Attack.

Used to export additional variables on an imported prefab. Stored Attack created using Attack.Create(AttackInfo) function.

◆ UseWorldSpace

bool AttackInfo.UseWorldSpace

Determines the parent for the spawned attack object.

Often used for projectiles, which travel independant of the AttackInfo.Owner.

True Attack will become a child of the owners parent.
False Owner will be set as the parent object.
