Detailed Description

Generic attack.

Attacks are defined as prefabs, then created from an AttackInfo template, using the Attack.Create(AttackInfo) function.

Conditions under which attack will destroy self.

Enumerator
Persistant 

No destroy conditions.

Timeout 

Destroy after set time.

Makes use of Attack.m_Duration and Attack.m_End.

OnHit 

Destroy on successful hit with a Entity.

OnCollide
