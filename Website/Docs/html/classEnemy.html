◆ m_Agent

NavMeshAgent Enemy.m_Agent
private

Agent used to generate navigation path.

Unity's built in pathfinding doesn't work too well with 2D scenes. Path travelling should be handled by the entity not by the agent. Utilizing built in transform updates results in unwanted changes to rotation and z position, rendering the entity invisible to the player.
